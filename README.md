# 엘리스의 깃허브 안내문 🐱💻

1. 📁 `01-*******, 02-********` 등의 이름을 가진 폴더명은  
   C언어를 배우며 만들어본 예제의 **순서**입니다.

2. 🔄 폴더 안에는 같은 예제를 계속 개선해 나가며  
   `ver.1`, `ver.2` 같은 이름으로 **새로운 버전**을 추가할 예정입니다.

3. 🌱 공유한 예제들은 **초보자가 만든 코드 예제**로,  
   공공의 목적 또는 **상업적 이용도 자유롭게 사용**하셔도 됩니다.

4. 🏷️ 출처 표기를 해주시면 감사하겠지만,  
   **꼭 하지 않으셔도 괜찮아요.** (부끄러워요.. 🙈)
